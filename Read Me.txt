In this programme, we use selenium for browser automation with Python, the same can also be done using Java only the sintax will be a bit different.

Basically we send request amazon for login O.T.P for the given number. The max no of times amazon allows to send otp is 20 so it is better to keep the frequency to 20(the no of times loop is to be run).
